# Frontend Directory

이 디렉토리는 **operato-wms-app** 프론트엔드 프로젝트가 위치할 곳입니다.

## 📁 예상 구조

```
frontend/
├── packages/
│   ├── operato-wes/
│   ├── operatofill/
│   └── metapage/
├── lerna.json
├── package.json
└── yarn.lock
```

## 🚀 다음 단계

1. **operato-wms-app** 프로젝트 내용을 이 디렉토리로 복사
2. 빌드 설정 확인 및 조정
3. 통합 빌드 테스트

## ⚠️ 주의사항

- 이 디렉토리는 `.gitignore`에서 `node_modules/`와 `dist/` 제외됨
- 빌드 산출물은 `src/main/resources/static/`으로 복사됨
