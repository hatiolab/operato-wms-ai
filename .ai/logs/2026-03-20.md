## 17:15 — PostgreSQL 문법 수정

### 작업 내용
- MySQL 함수를 PostgreSQL 문법으로 변경
- `DATE_SUB(CURRENT_DATE, INTERVAL 90 DAY)` → `CURRENT_DATE - INTERVAL '90 days'`
- del_flag 조건 통일: `(del_flag is null OR del_flag = false)` → `del_flag = 'N'`

### 변경/생성 파일
- `src/main/java/operato/wms/stock/service/InventoryTransactionService.java` — getDashboardAlerts 메서드 수정

### 주요 결정 또는 발견 사항
- PostgreSQL에서는 `DATE_SUB` 함수를 사용할 수 없음
- `CURRENT_DATE - INTERVAL '90 days'` 형식 사용
- del_flag 조건을 기존 패턴(`'N'`)과 일치시킴
