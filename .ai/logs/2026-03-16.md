## 22:40 — Seed SQL entity_columns name snake_case 변환 및 rank 시작값 변경

### 작업 내용
- entity_columns의 `name` 값을 camelCase에서 snake_case로 변환 (예: `rwaNo` → `rwa_no`, `dispositionType` → `disposition_type`)
- `rank` 필드 시작값을 0에서 10으로 변경 (10, 20, 30, ... 단위)

### 변경/생성 파일
- `src/main/resources/sql/V001__rwa_resource_seed.sql` — entity_columns INSERT 전체 수정

### 주요 결정 또는 발견 사항
- Operato WMS의 entity_columns.name은 DB 컬럼명(snake_case)을 사용해야 함
- rank 10 단위 증가 방식으로 향후 컬럼 삽입 여유 확보
